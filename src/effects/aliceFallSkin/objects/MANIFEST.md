# 爱丽丝掉落 · 装饰小图导出清单（Figma 24:104）

MCP 导出这些节点拿到的都是空白占位图，需要你在 Figma 里手动导出。

## 怎么做
1. 在 Figma 打开 `24:104`（✅爱丽丝2…）这帧。
2. 按下表的**节点ID / 位置**找到每个 `Object` 图层（dev 模式选中图层右栏会显示 node id）。
3. 选中 → 右键 Export / 右栏 Export 面板 → **PNG，2x**。
4. 按表里的**文件名**命名，存到本目录 `src/effects/aliceFallSkin/objects/` 下。
5. 放好后刷新页面即可自动生效（没放的会用 emoji 占位）。

> 透明背景的 PNG 最好；尺寸不必精确，代码会按下表的位置/大小摆放。
